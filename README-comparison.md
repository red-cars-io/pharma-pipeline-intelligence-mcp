## What You Get

The Pharma Pipeline Intelligence MCP fans out to 7 databases simultaneously and returns a composite **Pipeline Threat Score (0-100)** with investment verdicts:

- **Pipeline Threat Score** — how much competitive pressure a drug faces
- **TRL Assessment** — technology readiness level
- **Patent Exclusivity** — months until generic entry
- **Adverse Event Signals** — FDA FAERS divergence scoring
- **Regulatory Pathway** — FDA vs EMA comparison
- **Literature Momentum** — publication acceleration indicators

## Use Cases

### Biotech investment due diligence
`generate_pipeline_threat_report(drug_name="tirzepatide", indication="obesity")` → Pipeline Threat Score, clinical phase, patent expiry, adverse events, competitive landscape

### Competitive landscape analysis
`analyze_competitive_landscape(company="Eli Lilly", therapeutic_area="GLP-1")` → all competing drugs, phase distribution, threat score

### Patent cliff monitoring
`track_patent_exclusivity(drug_name="Humira", manufacturer="AbbVie")` → expiry date, biosimilar threat level, First-Mover Advantage score

## When to Choose Pharma Pipeline Intelligence MCP

**Choose this when:**
- You're an AI agent doing biotech investment research
- You need composite scores instead of raw data
- You want patent cliff and adverse event analysis in one call
- You're building competitive intelligence for pharma

**Choose ClinicalTrials.gov directly when:**
- You only need basic trial search
- You have a researcher doing manual analysis
- You don't need composite scoring or cross-database analysis
